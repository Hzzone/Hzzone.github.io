必备应用列表

Mac

开发

  • Postman: 模拟 HTTP 请求。
  • iterm2: Mac 默认 terminal 替代应用。
  • Visual Studio Code，Sublime Text 3: 代码编辑、阅读。
  • Cosbrowser: Mac 应用，浏览上传腾讯对象存储。
  • Dash: 各种 API 查询。

日常

  • 坚果云，Dropbox: 同步备份软件。
  • Skim，PDF Expert: PDF 阅读。
  • The Unarchiver: Mac 免费解压工具。
  • CheatSheet: 长按 Command 出现应用快捷键。
  • caffeine: Mac 保持屏幕常亮。
  • Paragon NTFS for Mac: Mac 支持 Windows NTFS 格式的硬盘。
  • Captuocr: Mac 截图 OCR 软件。
  • CleanMyMac: Mac 清理内存和缓存。
  • Quitter for Mac: 自动退出一段时间没有响应的应用。
  • CatchMouse: 多显示器之间快捷键切换。

提高生产力

  • mathpix: LaTeX 编辑，图片转 LaTeX，跨 Mac、Windows、Ubuntu、Android、iOS 的应用。
  • 精灵标注助手: 深度学习数据集标注。
  • iPic: 快速上传图片，生成 markdown 链接，支持新浪、腾讯云等图床。
  • Alfred: 快捷键，各种 workflow。

iOS

  • 熊猫吃短信: 垃圾短信过滤。
  • 万能播放器: 播放本地视频。
